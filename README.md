Ao aumentar o número de amostras n é esperado que a aproximação da integral seja mais precisa. Isso é garantido pela Lei dos Grandes Números. 

Arquivo "normal.py" calcula a integral da função densidade da distribuição normal entre -1 e 1 via Monte Carlo.

Arquivo "impropria_normal.py" calcula integral imprópria de 0 a infinito da função densidade da distribuição normal via Monte Carlo.
